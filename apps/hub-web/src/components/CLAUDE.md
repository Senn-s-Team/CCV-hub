成员清单
launch/: 启动弹窗子模块，负责路径浏览、启动参数、最近路径、移动端两段式流程与共享工具
AuthGate.tsx: 鉴权守门组件，处理登录态检查、口令登录、退出后回到登录页与已登录内容渲染
InstanceCard.tsx: 实例列表单元组件，以桌面行式与移动卡片同构展示项目名、路径、链接、状态、来源、最近心跳、启动时间与可用的打开、复制、停止动作
LaunchDialog.tsx: 启动弹窗容器组件，编排桌面/移动同构的路径/参数两段式启动向导、降级粘贴路径入口、字段错误、提交动作与弹窗生命周期
Toast.tsx: 轻量反馈组件，展示复制与启动成功的短消息
