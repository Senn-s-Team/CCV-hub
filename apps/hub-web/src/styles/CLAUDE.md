成员清单
tokens.css: 主题 token 契约，定义浅色、深色、系统主题、语义色、阴影、半径与内容宽度变量
base.css: 基础规则层，定义 reset、body、panel、surface 与全局排版基线
auth.css: 鉴权页面样式层，定义 AuthGate 登录卡片、检查态骨架与提交区域
controls.css: 通用控件层，定义 kicker、按钮、chip、旧主题控件与交互过渡规则
workbench-core.css: 工作台核心层，定义工具栏、实例舞台、实例卡片、空态、错误态、骨架屏与 Toast 基础样式
launch-dialog.css: 启动弹窗层，定义宿主机路径浏览、最近路径、启动参数、错误提示与提交栏样式
focus-motion.css: 交互动效层，定义统一 focus-visible、骨架动画、hover motion 与动效入口
responsive-core.css: 基础响应式层，定义平板、手机、bottom sheet 与 reduced-motion 规则
workbench-redesign.css: redesign 覆盖层，定义左侧状态驾驶舱、移动状态抽屉、抽象主题图形与浮动启动入口
